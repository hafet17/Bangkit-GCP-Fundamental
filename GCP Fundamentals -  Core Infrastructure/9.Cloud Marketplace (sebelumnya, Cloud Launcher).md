## **Cloud Marketplace (sebelumnya, Cloud Launcher)**
Misalnya, Anda ingin langkah cepat
untuk memulai GCP dengan mudah. Itulah keuntungan yang
diberikan oleh Google Cloud Launcher. Ini adalah fitur untuk menerapkan paket software fungsional
secara cepat di Google Cloud Platform. Anda tidak perlu mengonfigurasi
software, instance mesin virtual, penyimpanan, atau
setelan jaringan secara manual. Meski demikian,
jika mau, Anda bisa mengubah berbagai setelan sebelum meluncurkannya. Sebagian besar paket software di Cloud
Launcher tidak dikenakan tagihan tambahan di luar biaya penggunaan
normal untuk resource GCP. Beberapa gambar Cloud Launcher dikenakan tagihan biaya penggunaan, khususnya gambar yang dipublikasikan oleh pihak ketiga dengan software yang
berlisensi secara komersial. Tetapi, sebelum meluncurkannya,
semua perkiraan tagihan bulanan akan ditampilkan kepada Anda. Perlu diperhatikan bahwa
ini hanyalah perkiraan tagihan. Khususnya, fitur tidak
mencoba menaksir biaya jaringan karena biaya jaringan akan
bervariasi berdasarkan cara menggunakan aplikasi tersebut. Hal kedua yang perlu diperhatikan, GCP memperbarui gambar dasar untuk paket software ini guna memperbaiki masalah kritis & kerentanan. Tetapi GCP tidak mengupdate
software setelah diterapkan. Untungnya, Anda
akan memiliki akses ke sistem yang diterapkan, jadi Anda bisa mempertahankannya.
