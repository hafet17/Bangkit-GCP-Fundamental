## **API Open Source**
Banyak orang merasa takut
untuk menyelesaikan beban kerjanya di cloud karena khawatir akan
terkunci ke dalam vendor tertentu. Tapi melalui banyak cara, Google akan memberi pelanggan kemampuan
menjalankan aplikasinya di tempat lain, jika Google tidak lagi menjadi
penyedia terbaik bagi kebutuhan mereka. Berikut beberapa contoh
terkait cara Google membantu menghindari
kekhawatiran pelanggannya. Layanan GCP kompatibel
dengan produk open source. Misalnya, Cloud Bigtable, database yang
nanti akan kita bahas. Bigtable menggunakan antarmuka
database open source Apache HBase, yang memberi pelanggan
keuntungan portabilitas kode. Contoh lainnya, Cloud Dataproc
yang menawarkan lingkungan big data open source Hadoop
sebagai layanan yang dikelola. Google memublikasikan
elemen teknologi menggunakan lisensi
open source untuk membuat ekosistem yang memberi
pelanggan opsi selain Google. Misalnya, TensorFLow, library software open source untuk machine learning
yang dikembangkan di dalam Google, adalah inti dari ekosistem
open source yang canggih. Banyak teknologi GCP
memberikan interoperabilitas. Kubernetes memberi pelanggan
kemampuan untuk menggabungkan dan mencocokkan microservice yang
dijalankan di cloud yang berbeda, dan Google Stackdriver memungkinkan
pelanggan memantau beban kerja di beberapa penyedia cloud.
